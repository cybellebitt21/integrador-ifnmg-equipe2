# Referências Científicas — Projeto Integrador

## Objetivo

Este documento reúne artigos, pesquisas e materiais utilizados como base teórica para o desenvolvimento do projeto:

> Sistema de acompanhamento de plantas com monitoramento climático e umidade do solo utilizando IoT e visualização de dados.

---

# Contexto Regional

## As secas no Jequitinhonha

Link:
https://rbeur.anpur.org.br/rbeur/article/view/6164

Resumo:
O artigo aborda os impactos da seca no Vale do Jequitinhonha, destacando dificuldades de abastecimento hídrico e os desafios enfrentados pela agricultura familiar no semiárido mineiro.

Importância para o projeto:
Ajuda a contextualizar os problemas climáticos da região de Araçuaí e justificar a necessidade de tecnologias de monitoramento ambiental.

---

# IoT e Agricultura

## Protótipo IoT para monitoramento da umidade do solo

Link:
https://periodicos.ufes.br/bjpe/article/view/51114

Resumo:
Apresenta o desenvolvimento de um sistema IoT para monitoramento da umidade do solo utilizando sensores e transmissão de dados em tempo real.

Importância para o projeto:
Serve como referência técnica para arquitetura do sistema, sensores e monitoramento climático.

# Sensores e Irrigação Inteligente

## Smart Plant Monitoring and Irrigation System

Link:
https://arxiv.org/abs/2601.15830

Resumo:
Sistema inteligente de monitoramento ambiental e irrigação automatizada com sensores climáticos e visualização de dados.

Importância para o projeto:
Pode servir de inspiração para funcionalidades futuras do dashboard e recomendações automáticas.
