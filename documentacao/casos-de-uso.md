# Casos de Uso — AgroSensor

## Objetivo

Este documento apresenta os principais casos de uso do sistema AgroSensor, descrevendo as interações entre os usuários e a plataforma de monitoramento climático.

---

# Atores do Sistema

| Ator | Descrição |
|---|---|
| Usuário | Agricultor ou operador do sistema |
| Administrador | Responsável pelo gerenciamento do sistema |

---

# Casos de Uso

| Código | Caso de Uso | Descrição |
|---|---|---|
| UC01 | Realizar login | Permitir acesso ao sistema |
| UC02 | Visualizar dashboard | Exibir informações climáticas |
| UC03 | Visualizar temperatura | Mostrar temperatura ambiente |
| UC04 | Visualizar umidade do ar | Mostrar dados do sensor DHT22 |
| UC05 | Visualizar umidade do solo | Mostrar dados do sensor HW-080 |
| UC06 | Visualizar luminosidade | Mostrar dados do sensor LDR |
| UC07 | Consultar histórico | Permitir visualização de leituras anteriores |
| UC08 | Cadastrar usuário | Permitir criação de conta |
